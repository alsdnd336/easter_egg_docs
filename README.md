### 프로젝트 소개
깔끔한 UX/UI와 효율적인 매칭 시스템으로 기업과 마케팅 업체를 빠르고 정확하게 연결하는 매칭 플랫폼입니다.

- 개발기간: 2024.04 ~ 2024.06
- 개발 기여도: **90%**
- 기술 스택: **Flutter** **Spring Boot** **AWS** **MySQL**

### 👨‍💻 내가 기여한 부분
- 클라이언트 요청에 맞춘 앱 기획
- 디자인에 맞춰 전체 UI/UX 프론트엔드 구현(Flutter)
- 백엔드 API 설계 및 개발(Spring Boot / MyBatis / REST API)
- Android / iOS 앱 스토어 배포

### ⚙ 주요기능
<br>

**기업/마케팅업체 별 추천 기능 제공**

- 기업: 제품 카테고리 기반으로 전문 마케팅 업체 추천 및 추천 리스트 제공
- 마케팅업체: 전문 분야/역량에 맞는 제품 추천 및 제안 기회 제공
- 카테고리별 추천 업체 리스트로 탐색 효율을 강화

**역할 기반 맞춤 화면 제공**

- 기업/마케팅업체 별 전용 페이지 구성으로 필요한 기능에 빠르게 접근
- 제안 관리: 받은 제안 / 보낸 제안 리스트 제공
- 운영 기능: 상품(등록/삭제/수정) 관리 페이지 제공

**평점 시스템 구현**

- 객관적 비교를 위한 마케팅업체 평가 지표 제공

**부가기능**

- 알림 설정으로 주요 이벤트(추천/제안 등) 확인 편의성 강화
- 회원가입/비밀번호 변경/탈퇴 등 역할별 계정 관리 기능 제공
- 마케팅업체·제품을 빠르게 탐색할 수 있도록 검색 기능 제공

### 🛠 트러블 슈팅
<br>

🚨 **문제배경**

기획 단계에서 추천 기능 도입 요청이 있었으나, 추천 기준과 로직이 명확히 정의되지 않아 구현 방향 수립에 어려움이 있었습니다.

**💡 해결방안**

추천 기준을 카테고리와 평점으로 단순화하여, 사용자가 이해 가능한 형태의 추천 로직을 설계했습니다.
- 1차 필터링: 동일 카테고리 매칭
- 2차 정렬: 평점 높은 순 우선 추천
